# array.java
*an integer array wrapper*

---
## Usage
>
### new Array(size)
Create a new array with given size
>
### .get(index)
Get value at given index
>
### .push(value)
Add a value in the next unset array value

## Tools
>
### Run the demo
	sh run.sh
>
### Generate documentation
	sh docs.sh
>
### Run tests
	sh test.sh
